# AURYN Falante

Aplicativo offline para assistência de comunicação.

## Sobre o Projeto

AURYN Falante é um aplicativo desenvolvido para proporcionar uma solução de comunicação alternativa e aumentativa (CAA) de forma offline. O projeto visa auxiliar pessoas com dificuldades de comunicação verbal, oferecendo uma interface intuitiva e acessível que funciona sem necessidade de conexão com a internet.

O objetivo principal do AURYN Falante é promover a inclusão e autonomia comunicativa, permitindo que usuários possam expressar suas necessidades, sentimentos e ideias através de recursos visuais e sonoros.
